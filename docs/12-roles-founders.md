# 01 · Quiénes operamos Marketplace SMC

Tres founders. Marketplace SMC NO es un producto SaaS para terceros · es la herramienta operativa con la que nosotros tres operamos el negocio de comercio multi-canal.

# 02 · División de trabajo · cross-funcional

Los 3 founders compartimos logística y comercial. Guillermo es el único en parte técnica/plataforma. NO hay especialista en bodega ni especialista comercial · todos hacemos ambas cosas.

| Área | Camila | Javier | Guillermo |
|------|--------|--------|-----------|
| Logística (bodega · recepción · QA · despachos) | ✓ | ✓ | ✓ |
| Comercial (ventas · canales · clientes · sourcing) | ✓ | ✓ | ✓ |
| Plataforma técnica (sistema · IA · integraciones) | — | — | ✓ |
| Contabilidad + SII + galvarez.cl | — | — | ✓ |
| Estrategia + decisiones founders | ✓ | ✓ | ✓ |

# 03 · Camila Bianchi · Founder

**Foco**: logística + comercial · rotación según necesidad operativa.

- Recepción lotes desde Nexport · QA físico · conteo
- Gestión bodega (ubicación · etiquetado · picking)
- Operación canal Mercado Libre (listings · respuestas · disputas)

# 06 · Decisiones críticas

| Decisión | Quién decide | Cómo |
|----------|--------------|------|
| Sourcing producto nuevo | Los 3 (los 3 venden y operan) | Consenso |
| Pricing en canales | Los 3 | Consenso (puede ser ad-hoc rápido vía Slack) |
| Cambio plataforma | Guillermo · informa | Unilateral técnico |
| Compra a Nexport | Los 3 (monto · cantidad · plazos) | Consenso |
| Inversión nueva | Los 3 | Consenso |
| Tecnología subyacente | Guillermo | Unilateral · con info a los 3 |
| Conflictos | Mediación · arbitraje CAM si no se resuelve | Pacto socios |
